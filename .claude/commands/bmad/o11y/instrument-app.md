Add OpenTelemetry instrumentation to your application.

This command helps you:
1. Detect your application's language and framework
2. Configure auto-instrumentation or manual SDK setup
3. Set up proper resource attributes (service.name, environment, K8s metadata)
4. Configure exporters for your backend (Dynatrace, Grafana, Jaeger)
5. Validate instrumentation is producing correct telemetry

Supported languages: Node.js, Go, Python, Java, .NET

Usage: Describe your application (language, framework, deployment target) and desired signals (traces, metrics, logs).
